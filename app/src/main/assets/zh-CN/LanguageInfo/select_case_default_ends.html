MFP编程语言select，case，default和ends语句：

Select，case，default和ends构成了MFP中除if之外的另外一种条件程序块。他们的语法如下：

select expr

case value1

    ......

    break

case value2

    ......

    break

......

......

......

default

    ......

ends

。注意如果一个case块的结尾没有break语句，MFP将会执行下一个case块或者default块（如果下个条件分支为default）的语句。
