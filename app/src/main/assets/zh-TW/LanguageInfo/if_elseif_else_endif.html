MFP編程語言if，elseif，else和endif語句：

If，elseif，else和endif組成了MFP語言中的條件程式塊。它們的語法如下：

If conditionA

    ......

elseif conditionB

    ......

elseif conditionC

    ......

......

......

......

else

    ......

endif

。在上述例子中，conditionA，conditionB和conditionC均為運算式，它們的運算結果應該為布林值。如果它們運算值不是布林值，MFP將會將運算結果轉換為布林值。如果做不到這一點，MFP將報錯。
