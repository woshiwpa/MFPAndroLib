程式開發科學計算器：MFP編程語言手冊

在程式開發科學計算器中，除了內置函數，所有函數均使用一種叫做MFP的編程語言編寫。MFP的意思是Mathematical language For Parallel computing，意思是面向並行計算的數學語言。除了支持常用數學表達式和操作符，MFP語言還支持並行計算，面向對象編程，二進制數（格式為0b打頭，比如0b0011100），八進制數（格式為0打頭，比如0371.242），16進制數（格式為0x打頭，比如0xAF46BC.0DD3E），複數，陣列和矩陣，字串，函數，函數引用空間，變數，條件語句，循環語句，註釋和幫助等等。MFP語言對大小寫不敏感。

MFP編程語言包含以下語句：

function, endf, return

variable

if, elseif, else, endif

while, loop, do, until, for, next

break, continue

select, case, default, ends

try, throw, catch, endtry

class, public, private, self, super, this, endclass

call, endcall

citingspace, using

help, endh, @language

@compulsory_link

@execution_entry

@build_asset

在MFP語言中每一條語句可以占據一行或者多行。換句話說，如果一條語句太長，它可以被分割為几行，除了最后一行在每一行的末尾都跟隨有字符串" _"。例如，假設一個函數擁有十個參數，如果將函數聲明語句放在一行中將會非常的長：

function abcde(para1, para2, para3, para4, para5, para6, para7, para8, para9, para10)

。為了讓程序代碼更容易閱讀，我們可以把函數聲明語句分成3行，如下所示：

function abcde(para1, para2, para3, _

               para4, para5, para6, _

               para7, para8, para9, para10)

。并且我們依然可以在每一行的末尾添加注釋，在以上例子中，我們可以為每一行添加注釋如下：

function abcde(para1, para2, para3, _// 第一行有3個參數。

               para4, para5, para6, _		// 第二行也有三個參數。

               para7, para8, para9, para10) // 第三行有4個參數。

需要注意的是在MFP語言中每一行最多只能容納一條語句。不像C/C++，MFP語言不支持語句分隔號，比如";"。
