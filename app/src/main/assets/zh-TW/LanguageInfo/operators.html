程式開發科學計算器幫助：支持的運算運算子

運算運算子 定義 例子
=
 賦值操作符。本操作符將一個數值賦給一個變量。 x = 3 + 4將數值7賦給變量x。
==
 等於號。本符號意味著本符號左邊的表達式和符號右邊的表達式有同樣的數值。 x + 1 == 7 + 3意味著x + 1和7 + 3有同樣的數值，也就是10。通過這個表達式，我們可以解出x的值為9。
(
 左括號。位於一對括號中的表達式有更高的運算優先級。 x * (y + 3)
)
 右括號。位於一對括號中的表達式有更高的運算優先級。 x * (y + 3)
[
 左方括號。左方括號為一個數組的定義的開始。 [[1,2,3],[4,5,6]]定義了一個2*3矩陣，第一行為包括3個元素（也就是1，2和3）的數組，第二行也為包括3個元素（也就是4，5和6）的數組。該矩陣本身也為一個數組包括2個數組元素。
]
 右方括號。右方括號為一個數組的定義的結束。 [[1,2,3],[4,5,6]]定義了一個2*3矩陣，第一行為包括3個元素（也就是1，2和3）的數組，第二行也為包括3個元素（也就是4，5和6）的數組。該矩陣本身也為一個數組包括2個數組元素。
,
 逗號將數組中的每一個元素分隔開。 數組[2,3,4,5]有4個元素被3個逗號分隔開。
+
 加號（支援復數，矩陣和字串） 2.3 + 4.6 + 2i = 6.9 + 2i
[1, 2] + [3, 4] = [3, 6]
[1, 2] + " hello" = "[1,2] hello"
-
 減號（支援復數和矩陣） 4.6 - 2.3 = 2.3
[1, 2] - [3, 4] = [-1, -1]
*
 乘號，支援復數和矩陣，比加號和減號優先級高。 1 + 2 * 3i = 1 + 6i
[[1, 2]] * [[3], [4]] = [[11]]
/
 除號，支援復數和矩陣，比加號和減號優先級高。 6 - 3 / 2i = 6 + 1.5i
[[10,0.5]]/[[1,2],[3,4]] = [[-19.25,9.75]]
\
 左除號（主要用於矩陣相除，比如計算Ax=b中的x時，x=A\b，註意第一個操作數是除數，第二個是被除數，如果除數不是矩陣，和除號的功能完全一樣）。 6 - 3 / 2i = 6 + 1.5i
[[1,2],[3,4]]\[[11],[32]] = [[10], [0.5]]
&
 位與，比加減乘除的優先順序高。注意位與只接受非負運算數，並且將自動把運算數取整。 4 + 5.2 & 3.7 = 5
|
 位或，比加減乘除的優先順序高。注意位或只接受非負運算數，並且將自動把運算數取整。 2.8 / 5.2 | 3.7 = 7
^
 位異或，比加減乘除的優先順序高。注意位異或只接受非負運算數，並且將自動把運算數取整。 14.2 / 5.2 ^ 3.7 = 2.3667
**
 次方，比加減乘除和所有的位運算子優先順序高。註意如果兩個運算數都可以為復數。 2 * 4 ** 3 = 128
(-4) ** 0.5 = 2i
3**(2+i) = 4.0935 +8.0152*i
+
 正號，比所有的二元運算子優先順序高。 4 - +3 = 1
+[4,3] = [4, 3]
-
 負號，比所有的二元運算子優先順序高。 4 ** -1 = 0.25
-1 ** 4 = 1
-[3.71i, [4, 5.33-6i]] = [-3.71*i,[-4, -5.33+6*i]]
%
 百分號，比所有的二元運算子和正負號優先順序高。 -401.78% = -4.0178
5.77% = 0.0577
!
 否，比所有的二元運算子和正負號優先順序高。如果操作數為0，返回True,如果操作數非0，返回False。注意其操作數必須為實數。 !-0.2 = False
~
 位非，比所有的二元運算子和正負號以及否的優先順序高。注意它僅接受非負運算數。如果運算數不是整數，將自動轉換為整數。 ~0 = -1
~0.1 = -1
~9 = -10
!
 階乘號，比所有二元運算子以及一元左置運算子優先順序高。注意它僅接受非負運算數。如果運算數不是整數，將自動轉換為整數。 7! = 5040
'
 一維二維矩陣或者單一元素的轉秩，比所有的二維操作符和左置一維操作符的優先順序高。注意它能將一維矩陣轉換為二維矩陣但如果操作數是單一元素，則返回操作數的原始值。 (3 + 4i)' = (3 + 4i)
[1, 2+3i]' = [[1], [2+3i]]
[[1, 2],[3,4]] = [[1,3], [2, 4]]
